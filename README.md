# 🌐 Muhammadxoja's Portfolio

Welcome to my personal portfolio! This project is a reflection of my web development journey, showcasing modern design principles with elegant parallax effects, smooth animations, and a personalized "My Like Musics" section integrated with a MySQL database.

![Portfolio Preview](https://muhammadxoja.netlify.app/preview.png)

## ✨ Features
- Fully responsive and optimized for all devices
- Captivating parallax scrolling and smooth animations
- "My Like Musics" section powered by MySQL for personalized music collection
- SEO optimization to enhance search visibility
- Unique starry night parallax effect in the footer for a mesmerizing experience

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: MySQL (for "My Like Musics" section)
- Deployment: Netlify

## ⚙️ Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open the `index.html` file directly in your browser.

### Database Setup (Optional)
If you want to use the "My Like Musics" section with a database:
1. Set up a MySQL database.
2. Import the provided SQL file.
3. Configure the backend to connect to your database.
